# USD/CNH 和 布伦特原油价格监控系统

## 系统架构
```
├── config.py              # 配置文件（API密钥等）
├── price_monitor.py       # 主监控脚本
├── telegram_bot.py        # Telegram推送模块
├── data_collector.py      # 数据采集模块
├── requirements.txt       # Python依赖
└── setup_guide.md        # 设置指南
```

## 免费API选择
### 1. Alpha Vantage（推荐）
- **USD/CNH**: `https://www.alphavantage.co/query?function=CURRENCY_EXCHANGE_RATE&from_currency=USD&to_currency=CNH&apikey={KEY}`
- **布伦特原油**: `https://www.alphavantage.co/query?function=GLOBAL_QUOTE&symbol=BRNT&apikey={KEY}`
- 免费限制：5次/分钟，500次/天

### 2. 注册步骤：
1. 访问 https://www.alphavantage.co/support/#api-key
2. 填写邮箱和密码注册
3. 获取免费API密钥

## Telegram机器人设置
### 创建机器人：
1. 在Telegram中搜索 @BotFather
2. 发送 `/newbot` 创建新机器人
3. 设置机器人名称和用户名
4. 获取API Token（格式：`1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`）

### 获取Chat ID：
1. 私聊你的机器人发送任意消息
2. 访问：`https://api.telegram.org/bot{TOKEN}/getUpdates`
3. 查看JSON中的 `chat.id` 字段

## Python脚本功能
### 核心功能：
- **定时获取**：每5分钟获取一次最新价格
- **价格对比**：对比上次价格，计算涨跌幅
- **阈值报警**：设置汇率/油价触发阈值
- **推送通知**：通过Telegram发送格式化消息

### 推送消息格式：
```
📈 价格监控报告 📉

💰 **USD/CNH**: 7.1234
    📊 涨跌幅: +0.15% (较上次 7.1123)
    ⏰ 更新时间: 14:30:05

🛢️ **布伦特原油**: $85.67/桶
    📊 涨跌幅: -0.42% (较上次 $86.03)
    ⏰ 更新时间: 14:30:05

📈 **提醒设置**:
    - USD/CNH > 7.15 或 < 7.10 时通知
    - 原油价格波动 > 1% 时通知

🔄 下次检查：14:35:05
```

## 部署选项
### 1. 本地电脑运行（推荐初学者）
- 24小时开机
- 使用 `crontab` 或 `schedule` 库定时执行

### 2. 云服务器/VPS
- 阿里云/腾讯云轻量应用服务器
- 月费约5-15元

### 3. 免费云函数
- Cloudflare Workers（有速率限制）
- Railway/Heroku免费额度

## 开发步骤
1. 注册Alpha Vantage获取API密钥
2. 创建Telegram机器人获取Token
3. 安装Python环境（3.8+）
4. 安装依赖：`requests`, `python-telegram-bot`, `schedule`
5. 配置config.py文件
6. 测试数据获取和推送
7. 设置定时任务

## 技术栈
- Python 3.8+
- requests (HTTP请求)
- python-telegram-bot (Telegram推送)
- schedule (定时任务)
- logging (日志记录)
